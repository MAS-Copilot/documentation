设备管理信息数据库 (EquipmentManagement)

设备状态信息数据库的结构表规划：

设备信息表 (Equipment)

字段名称 数据类型 描述
EquipmentID INT 设备唯一标识符，主键，自增
EquipmentName VARCHAR(255) 设备名称
EquipmentType VARCHAR(100) 设备类型
SerialNumber VARCHAR(255) 设备序列号，唯一
InstallationDate DATE 设备安装日期
LastMaintenanceDate DATE 最后一次维护/检修日期
WarrantyStartDate DATE 保修期开始日期
WarrantyEndDate DATE 保修期结束日期
Status VARCHAR(50) 设备当前状态（运行、停机、维修中等）
Manufacturer VARCHAR(255) 制造商
Model VARCHAR(255) 设备型号
Location VARCHAR(255) 设备安装位置
Notes TEXT 设备备注信息
CreatedDateTime DATETIME 记录创建时间
LastModifiedDateTime DATETIME 记录最后修改时间

设备用户信息(EquipmentUsers)

字段名称 数据类型 描述
UserID INT 用户的唯一标识符，主键
Username VARCHAR(255) 用户名，唯一
PasswordHash VARBINARY(64) 用户密码的哈希值
PasswordSalt VARBINARY(64) 密码盐，用于加密用户密码
LoginCount INT 用户登录次数
LastLoginTime DATETIME 最后一次登录时间
Role ENUM('MAS', 'Admin', 'Engineer', 'Operator') 用户权限，枚举类型
RegistrationDate DATETIME 注册日期

用户设置(EquipmentUserSettings)

字段名称 数据类型 描述
SettingID INT 设置的唯一标识符，主键
UserID INT 用户唯一标识符，外键，关联到EquipmentUsers
Theme VARCHAR(100) 用户选择的主题
Language VARCHAR(100) 用户选择的语言
TimeZone VARCHAR(100) 用户选择的时区
NotificationPreference VARCHAR(100) 用户选择的通知偏好
ItemsPerPage INT NOT NULL 每页显示的条目数量

设备产品信息表 (EquipmentProduct)

字段名称 数据类型 描述
ProductID INT 产品唯一标识符，主键
EquipmentID INT 设备唯一标识符，外键，关联到Equipment
ProductName VARCHAR(255) 产品名称
ProductType VARCHAR(100) 产品类型
ProductDescription TEXT 产品描述
ProductModel VARCHAR(255) 产品型号
ProductSerialNumber VARCHAR(255) 产品序列号，唯一
Productsource VARCHAR(20) 产品来源，标识系统录入还是手动
ProductExpiryDate DATE 产品有效日期，仅日期部分，无时间
CreatedDateTime DATETIME 记录创建时间

设备每小时产量记录表 (EquipmentHourlyProductionRecord)

字段名称 数据类型 描述
HourlyProductionRecordID INT 每小时产量记录的唯一标识符，主键
EquipmentID INT 设备唯一标识符，外键，关联到Equipment表
RecipeID VARCHAR(100) 配方ID
ProductionCount INT 该小时的产量数量
DefectiveCount INT 该小时的不良品数量
ProductionDateTime DATETIME 该小时的产量记录生成的时间戳
Notes TEXT 关于此次每小时产量记录的额外注释或详细信息

设备生产记录表 (EquipmentProductionRecord)

字段名称 数据类型 描述
ProductionRecordID INT 生产记录的唯一标识符，主键
EquipmentID INT 设备唯一标识符，外键，关联到Equipment表
ProductionDate DATE 生产记录的日期
Shift VARCHAR(50) 生产班次（早班、晚班）
ProductionQuantity INT 在该日期班次的总产量
TargetQuantity INT 目标产量
ProductionTime DECIMAL(10,2) 实际生产时间（小时）
Downtime DECIMAL(10,2) 停机时间（小时）
CycleTime DECIMAL(10,2) 平均生产节拍时间（秒/单位）
Efficiency DECIMAL(5,2) 生产效率（百分比）
TimeBasedEfficiency DECIMAL(5,2) 基于时间的生产效率（百分比）
QuantityBasedEfficiency DECIMAL(5,2) 基于产量的生产效率（百分比）
CombinedEfficiency DECIMAL(5,2) 综合效率（百分比）
Quality DECIMAL(5,2) 良品率（百分比）
Notes TEXT 生产记录的额外注释或详细信息

设备状态记录表 (EquipmentStatusRecord)

字段名称 数据类型 描述
StatusRecordID INT 状态记录的唯一标识符，主键
EquipmentID INT 设备唯一标识符，外键，关联到Equipment表
RecordDateTime DATETIME 状态记录生成的时间戳
HaltCount INT 设备在该记录时段内的停机次数
FaultCount INT 设备在该记录时段内的宕机次数
AlarmCount INT 设备在该记录时段内的报警次数

设备停机事件表 (EquipmentHaltEvent)

字段名称 数据类型 描述
HaltEventID INT 停机事件的唯一标识符，主键
StatusRecordID INT 设备唯一标识符，外键，关联到EquipmentStatusRecord表
StartTime DATETIME 停机开始的时间戳
EndTime DATETIME 停机结束的时间戳（如果尚未恢复，则为NULL）
ReasonForHalt VARCHAR(255) 停机原因描述
ReportedBy VARCHAR(100) 报告停机事件的操作员或维护人员姓名
Notes TEXT 关于此次停机事件的额外注释或详细信息

设备宕机事件表 (EquipmentFaultEvent)

字段名称 数据类型 描述
FaultEventID INT 宕机事件的唯一标识符，主键
StatusRecordID INT 状态记录的唯一标识符，外键，关联到EquipmentStatusRecord表
StartTimestamp DATETIME 宕机开始的时间戳
EndTimestamp DATETIME 宕机结束的时间戳（如果尚未恢复，则为NULL）
FaultDescription VARCHAR(255) 宕机故障描述
SeverityLevel TINYINT 故障严重等级（1-紧急、2-重要、3-一般）
RepairActionTaken VARCHAR(255) 采取的修复行动记录
ResolvedBy VARCHAR(100) 解决宕机问题的维护人员姓名或团队
ReplacementParts VARCHAR(255) 更换的零部件信息（如有）
CostOfRepair DECIMAL(10,2) 维修成本（如果已知）
Notes TEXT 关于此次宕机事件的额外注释或详细信息

设备报警事件表 (EquipmentAlarmEvent)

字段名称 数据类型 描述
AlarmEventID INT 报警事件的唯一标识符，主键
StatusRecordID INT 设备唯一标识符，外键，关联到EquipmentStatusRecord表
AlarmTimestamp DATETIME 报警发生的时间戳
AlarmType VARCHAR(100) 报警类型（如温度过高、压力过大、部件故障等）
AlarmDescription VARCHAR(255) 报警详细描述
AcknowledgedBy VARCHAR(100) 确认报警的操作员或维护人员姓名
ResolvedTimestamp DATETIME 报警解决的时间戳（未解决为NULL）
ResolutionDetails TEXT 解决报警问题的具体措施和结果
AlarmSeverity TINYINT 报警严重等级（1-紧急、2-重要、3-一般）
Notes TEXT 关于此次报警事件的额外注释或详细信息

产能利用率统计表 (CapacityUtilization)

字段名称 数据类型 描述
UtilizationID INT 产能利用率统计的唯一标识符，主键
EquipmentID INT 设备唯一标识符，外键，关联到Equipment表
Hour TINYINT 时间段（小时）
ProductionTimeMinutes SMALLINT 实际生产时间（分钟）
Quantity SMALLINT 产量
RecordTime DATETIME 记录的时间戳
Notes TEXT 关于此次记录的额外注释或详细信息

设备周期时间统计表(EquipmentCycleTime)

字段名称 数据类型 描述
CycleTimeID INT 周期时间统计的唯一标识符，主键
EquipmentID INT 设备唯一标识符，外键，关联到Equipment表
Value1 float 工位1 CT值
Value2 float 工位2 CT值
Value3 float 工位3 CT值
Value4 float 工位4 CT值
Value5 float 工位5 CT值
Value6 float 工位6 CT值
Value7 float 工位7 CT值
Value8 float 工位8 CT值
Value9 float 工位9 CT值
Value10 float 工位10 CT值
RecordTime DATETIME 记录的时间戳
Notes TEXT 关于此次记录的额外注释或详细信息
