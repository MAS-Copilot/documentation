CheckPlcConnection()

检查PLC连接状态

bool CheckPlcConnection()

Returns

bool

连接活跃返回true，否则返回false

Remarks

lsConnected属性获取客户端套接字在最后一次I/O操作时的连接状态。它返回false时，客户端套接字要么从未连接，要么不再连接。 由于PlC的IsConnected属性仅反映截至最近操作的连接状态，因此应该尝试发送或接收消息以确定当前状态。
